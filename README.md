# Diwali Sales Analysis

## Overview

This project focuses on analyzing **Diwali sales data** using **Python** to understand customer behavior and sales trends during the festive season. The goal is to extract meaningful insights that can help businesses improve sales strategies and customer targeting.

This is a beginner-friendly data analytics project, suitable for freshers and entry-level roles.

---

## Dataset

* **Name:** Diwali Sales Dataset
* **Format:** CSV
* **Description:** Contains customer demographics, product categories, and purchase details during Diwali sales.
* **Source:** Public / Sample dataset

---

## Tools & Technologies

* **Python**
* **Libraries Used:**

  * pandas
  * numpy
  * matplotlib
  * seaborn
* **Environment:** Jupyter Notebook / VS Code

---

## Steps Performed

1. Loaded the dataset using Python (pandas)
2. Performed Exploratory Data Analysis (EDA)
3. Cleaned the data (handled missing values, corrected data types)
4. Analyzed sales based on:

   * Gender
   * Age group
   * State
   * Occupation
   * Product category
5. Created visualizations to identify trends and patterns

---

## Key Insights

* Identified top purchasing customer groups during Diwali
* Found high-performing product categories
* Observed sales trends across different states and age groups

---

## How to Run the Project

1. Ensure the following files are present in the project folder:

   * `Diwali Sales Data.csv`
   * `Diwali_Sales_Analysis.ipynb`

2. Install required Python libraries

   ```bash
   pip install pandas numpy matplotlib seaborn
   ```

3. Open Jupyter Notebook

   ```bash
   jupyter notebook
   ```

4. Run `Diwali_Sales_Analysis.ipynb` step by step to view the analysis and visualizations
